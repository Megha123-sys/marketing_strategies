# Marketing Strategies Hub

A modern React website built with Next.js, Tailwind CSS, and Framer Motion.

## Features

- ✅ Sidebar navigation with 6 main sections
- ✅ Top navbar with search bar and Subscribe button
- ✅ All pages as React components
- ✅ Tailwind CSS styling with Midnight Blue and Teal theme
- ✅ Framer Motion animations
- ✅ Article data stored in `data/articles.js`
- ✅ White cards with rounded corners
- ✅ Mobile-responsive design

## Pages

- **Home** (`/marketing`) - Featured articles and recent content
- **Strategy Hub** (`/marketing/strategy-hub`) - Marketing strategies and frameworks
- **Case Studies** (`/marketing/case-studies`) - Real-world campaign examples
- **Tools** (`/marketing/tools`) - Essential marketing tools
- **Insights** (`/marketing/insights`) - Industry trends and analysis
- **About Me** (`/marketing/about`) - Personal profile and contact

## Access

Navigate to: `http://localhost:3000/marketing`

## Theme Colors

- **Midnight Blue**: `#0F172A` (primary dark)
- **Teal**: `#14B8A6` (accent color)
- **White Cards**: Clean, modern card design with rounded corners

## Data Structure

Articles are stored in `data/articles.js` with fields:
- `id` - Unique identifier
- `title` - Article title
- `category` - Article category
- `summary` - Short description
- `date` - Publication date
- `readTime` - Estimated reading time
- `content` - Full article content (markdown)
