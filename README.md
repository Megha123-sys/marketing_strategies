# Meta Ads Dashboard for UpGrad Study Abroad

A comprehensive marketing control center for visualizing, testing, and optimizing Meta ad campaigns for EdTech brands.

## Features

- **Campaign Tracker**: Real-time metrics (CTR, CPA, ROAS, Impressions, Spend, Leads)
- **Creative A/B Testing**: Upload and compare creative performance
- **Geo Performance**: Regional breakdown and insights
- **Budget Allocator**: Interactive budget distribution tool
- **Lead Source Integration**: Connect to CRM systems
- **Ad Copy Suggestion**: AI-powered ad copy generation
- **Automated Recommendations**: Daily optimization suggestions
- **Export Options**: CSV and Google Sheets export

## Tech Stack

- **Framework**: Next.js 14 (React)
- **Styling**: Tailwind CSS
- **Charts**: Recharts
- **Icons**: Lucide React
- **Animations**: Framer Motion

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
npm start
```

## Environment Variables

Create a `.env.local` file:

```
NEXT_PUBLIC_OPENAI_API_KEY=your_openai_api_key_here
NEXT_PUBLIC_META_APP_ID=your_meta_app_id_here
NEXT_PUBLIC_META_APP_SECRET=your_meta_app_secret_here
```

## Deployment

The project is ready for deployment on Vercel:

1. Push to GitHub
2. Import project to Vercel
3. Add environment variables
4. Deploy
